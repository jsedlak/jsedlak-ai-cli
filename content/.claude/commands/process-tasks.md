description: Process tasks for a given PRD

Use `.ai/process-tasks.md` to process all tasks associated with a PRD. The user will pass you the name of a PRD, located in `.prd/`. Process the tasks in any `tasks.md` or `[task-name]-tasks.md` within the `.prd/[prd-name]/` folder
